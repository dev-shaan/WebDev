# TinDog Project

TinDog is a responsive landing page for a fictional dog dating app built with Bootstrap. This was my first Bootstrap project and I made it while learning frontend development.

## About The Project

The page is a single-page marketing site with a hero section, feature cards, a testimonial area, pricing cards, and a footer. It was created to practice Bootstrap layout classes, responsive design, and reusable UI sections.

## Features

- Responsive layout built with Bootstrap 5
- Hero section with download buttons
- Feature highlights with custom icons
- Testimonial section
- Pricing cards with multiple plans
- Footer with newsletter form and social links

## Built With

- HTML5
- CSS3
- Bootstrap 5

## Project Structure

- [index.html](index.html) - Main page content and Bootstrap layout
- [css/style.css](css/style.css) - Custom styles and visual tweaks
- [images/](images) - Project images and assets

## Getting Started

To view the project locally, open [index.html](index.html) in your browser.

If you want to run it from a local server, you can use any simple static server or the Live Server extension in VS Code.

or just copy the code and paste it into an HTML file and open it in your browser.
