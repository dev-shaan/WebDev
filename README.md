# WebDev (advaced HTML + CSS) projects 🚀

A collection of frontend development projects and experiments built with HTML, CSS, JavaScript, and Bootstrap. This repository serves as a portfolio demonstrating core web development skills, including responsive design, layout techniques (Flexbox), and component implementation.

## 📁 Project Structure

This repository contains several individual projects:

- **TinDog**: A responsive landing page built with **Bootstrap**. Demonstrates grid systems, navigation bars, and responsive components.
- **flexbox_pricing**: A pricing table layout showcasing advanced **CSS Flexbox** techniques for alignment and spacing.
- **basic projects**: A folder containing foundational exercises and smaller scripts to practice core web concepts.
- **.vscode**: Visual Studio Code workspace settings and extensions configured for this project.

## 🛠️ Technologies Used

- HTML5
- CSS3 (including Flexbox and Grid)
- Bootstrap 5
- Git & GitHub

  ## Overall Progress

These projects show my learning path from simple HTML practice to more polished, responsive frontend pages. I worked through basic structure, then moved into styling with CSS, then into layout systems like Grid and Flexbox, and finally into animations and Bootstrap-based design.

## Repository Structure
- `basic projects/` for small standalone mini-projects
- `flexbox_pricing/` for a responsive pricing table
- `TinDog (Bootstrap) Project/` for a Bootstrap website clone
